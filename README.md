# Обработка текстовой информации

Мой первый мини проект на языке программирования C++.

**Тех. задание:**
1. Чтение файла Input.txt построчно
2. Оформатирование текста: 
    - Удаление лишних, пробелов, запятых, двоеточий, точек с запятой, восклицательных и вопросительных знаков (Исключение: тире)
    - Строка не должна быть больше 60 символов, иначе она не выводится
    - Количество строк не должно быть больше 10, иначе другие не учитываются
3. Считывание количества символов в полученном тексте
4. Подсчет количества строк в полученнном тексте
5. Запись отформатированного текста в Output.txt

6. Найти и вывести слова содержащие указанное слово. К примеру, при, яблок и т.п.
7. Вывести и занести в файл Output-line.txt слова, состоящие из двух и более слогов (признаком слога является наличие гласной буквы)
